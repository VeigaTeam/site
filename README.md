# 🏋️‍♂️ Veiga Team - Academia de Artes Marciais

Site moderno e responsivo para a academia de artes marciais Veiga Team, com foco em performance, SEO e experiência do usuário.

## 🚀 Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Performance**: Service Worker, Lazy Loading, PWA
- **SEO**: Meta tags otimizadas, Open Graph, Schema.org
- **Acessibilidade**: Navegação por teclado, contraste alto, redução de movimento
- **Design**: Flexbox, Grid, Animações CSS, Responsivo

## 🚀 Instalação e Uso

### **Requisitos**
- Servidor web (Apache, Nginx, ou servidor local)
- Navegador moderno com suporte a ES6+

### **Instalação Rápida**
1. Clone o repositório:
```bash
git clone https://github.com/rveiga08/site.git
cd site
```

2. Abra o arquivo `index.html` em um servidor web

3. Para desenvolvimento local, use um servidor:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

## ⚙️ Configuração

### **Personalização**
Edite o arquivo `js/config.js` para:
- Alterar informações da academia
- Configurar endpoints de API
- Definir IDs de analytics
- Ajustar configurações de performance

### **SEO e Meta Tags**
- Atualize as meta tags no `<head>` do HTML
- Configure URLs canônicas
- Ajuste Open Graph para redes sociais

### **Service Worker**
- O Service Worker é registrado automaticamente
- Cache offline habilitado por padrão
- Estratégias de cache configuráveis
